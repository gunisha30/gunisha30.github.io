# Gunisha Chaturvedi Portfolio

Free static portfolio site for GitHub Pages.

## Publish
1. Create a GitHub repo named `gunisha-portfolio` or `<your-github-username>.github.io`.
2. Upload `index.html`, `styles.css`, `script.js`, `Image.jpeg`, and `Resume_Gunisha_Chaturvedi.docx` to the repo root.
3. Go to **Settings > Pages**.
4. Select **Deploy from branch**, branch `main`, folder `/root`.
5. Save and wait for GitHub to publish the site.
